# d2-icon

本框架集成了 font awesome字体图标, 建议所有后台图标用上面的图标

# 优点

  • Font Awesome为您提供可缩放的矢量图标，您可以使用CSS所提供的所有特性对它们进行更改，包括：大小、颜色、阴影或者其它任何支持的效果。
  • 集中管理，方便效率
  • 样式统一

# 缺点

  • 需要去了解技术，增加学习成本

# 使用方法


// 这样用没有毛病 但是也没什么用
<d2-icon/>

// 指定图标名称
<d2-icon name="github"/>

// 设置行内样式
<d2-icon name="github" style="font-size: 100px;"/>

// 设置 class
<d2-icon name="github" class="icon-class-demo"/>
